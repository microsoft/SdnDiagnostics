# Install-SdnDiagnostic

## SYNOPSIS
Install SdnDiagnostic Module to remote computers if not installed or version mismatch.

## SYNTAX

```
Install-SdnDiagnostic [-ComputerName] <String[]> [[-Credential] <PSCredential>] [-Force] [<CommonParameters>]
```
